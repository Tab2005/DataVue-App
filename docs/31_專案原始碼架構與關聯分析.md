## 核心模組關聯與依賴關係

DataVue 維持前後端分離架構：後端以 FastAPI 模組化路由、服務與資料存取層組成；前端以 React Router 頁面、功能元件、hooks 與 service client 組合。

### 後端架構

- `backend/main.py` 掛載核心 router 與模組 router。
- `backend/modules/*` 是目前主要產品功能邊界，Meta Andromeda、GA4、Contribution 已逐步拆成 repository/service/router/schema 或更細分子模組。
- `backend/core/*` 承載設定、排程、啟動、例外與安全等跨模組基礎設施。
- `backend/database/models/*` 是 SQLAlchemy model，`backend/alembic/versions/*` 是 migration 歷史。

### 前端架構

- `frontend/src/pages/*` 是路由頁面與組合層。
- `frontend/src/components/*` 是可重用 UI 或功能 panel；Phase D 後 `Contribution` 與 `MetaAndromeda/Monitoring*` 已拆成多個小元件。
- `frontend/src/hooks/*` 管理跨頁面或大型頁面的狀態與 side effect。
- `frontend/src/services/*` 封裝 API client。
